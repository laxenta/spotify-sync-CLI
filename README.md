# ColorWall 
## By Laxenta Inc.
4K Wallpaper Engine - Download stunning wallpapers from 7+ sources

## Features
- 4K image (depends on source)
- caching system (needs work)
- Scrapes from 7 different wallpaper sites
- Clean UI built with Next.js + Electron
- Standalone desktop app

## Installation
Download the latest release (450MB)
Run the installer
Enjoy premium wallpapers
